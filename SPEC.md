# 宏观仪表盘 SPEC.md

## 1. 项目概述

**项目名称**：Macro Dashboard（宏观仪表盘）
**项目类型**：数据仪表盘 + API 服务
**核心功能**：中国视角全球大类资产宏观量化展示，数据全部来自公开源
**目标用户**：投顾从业者、机构投资者

---

## 2. 技术架构

```
┌─────────────────────────────────────────────────────────────┐
│  前端（Vercel 静态部署）                                      │
│  React + ECharts                                             │
│  读取 Render API + GitHub Raw JSON                           │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  后端（Render FastAPI）                                      │
│  - FRED API 拉取美国宏观数据                                  │
│  - yfinance 拉取全球资产价格                                  │
│  - 从 GitHub Raw 读取中国宏观数据 JSON                        │
│  - 计算信号 + 观点映射                                        │
│  - SQLite 存储历史数据                                        │
└─────────────────────────────────────────────────────────────┘
                              ▲
                              │
┌─────────────────────────────────────────────────────────────┐
│  本地脚本（用户本地 Windows 机器）                             │
│  - AKShare 拉取中国宏观数据                                   │
│  - 生成 JSON push 到 GitHub                                  │
│  - 每日 08:30 自动触发（或手动跑）                            │
└─────────────────────────────────────────────────────────────┘
```

## 4. 页面结构（7大模块）

### 4.1 顶部状态栏
- RISK ON/OFF 徽章 + 置信度
- 主要驱动因素文字

### 4.2 模块一：宏观概览
- VIX 图表（面积图）
- 北向资金 3 日累计
- 风险偏好评分

### 4.3 模块二：中国内核
- PMI 仪表盘
- CPI/PPI 趋势图
- M1-M2 剪刀差
- 社融增量

### 4.4 模块三：全球宏观
- ISM PMI
- 核心 PCE
- 美联储资产负债表趋势

### 4.5 模块四：美元与流动性
- DXY 美元指数
- 中美利差（10Y）
- USD/CNH

### 4.6 模块五：全球核心资产
- 标普 500（K线/折线）
- 沪深 300
- 黄金、原油
- 10Y 美债收益率

### 4.7 模块六：波动率与风险
- 高收益债利差
- MOVE 指数
- A股隐含波动率

## 5. 设计风格

参考 ztrader.ai "黑金终端"风格：
- **主色**：深色背景（#0a0e17 / #111827）
- **强调色**：金色（#f0b90b / #ffd700）
- **辅助色**：绿色（#00d26a 看多）、红色（#f6465d 看空）
- **字体**：JetBrains Mono（数字）、Inter（正文）
- **图表**：深色主题，网格线淡化
- **布局**：高密度信息卡片，2-3列网格

## 6. API 设计

### GET /api/v1/overview
返回顶层信号 + 核心指标

### GET /api/v1/china
返回中国宏观数据

### GET /api/v1/global
返回全球宏观数据

### GET /api/v1/assets
返回全球资产价格

### GET /api/v1/risk
返回波动率与风险指标
