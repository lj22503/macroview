以下是一份**面向小白的字段解释文档**，涵盖API返回的所有核心字段。每个字段都用了最通俗的语言解释，并附带"打个比方"帮助理解。

---

## 📖 宏观仪表盘字段解释（小白版）


### 一、顶层概览（overview）—— 仪表盘的"结论页"

这一部分是整个仪表盘的**结论摘要**，告诉用户"现在市场整体是什么情况，我该怎么做"。


| 字段 | 含义（一句话） | 打个比方 | 取值范围 |
|---|---|---|---|
| **bias** | 当前市场应该"进攻"还是"防守" | 就像开车时的红绿灯：**RISK_ON**=绿灯（可加速），**RISK_OFF**=红灯（踩刹车），**NEUTRAL**=黄灯（减速观察） | `RISK_ON` / `NEUTRAL` / `RISK_OFF` |
| **confidence** | 这个判断有多大的把握 | 就像天气预报说"明天80%概率下雨"——这个数越高，越值得相信当前的信号 | 0-100 的整数 |
| **score** | 宏观环境的综合评分 | 就像一个学生的综合考试成绩：**正分**=环境好，**负分**=环境差。**+1.5以上**=优秀，**-1.5以下**=不及格 | -2.0 到 +2.0 |
| **primary_driver** | 今天市场上涨/下跌的主要原因 | 就像医生诊断时说"你发烧主要是因为嗓子发炎"——告诉你需要盯住什么因素 | 字符串列表 |
| **narrative** | 用一句人话告诉你现在的宏观状态 | 就像朋友跟你聊天时总结："最近经济在回暖，可以多买点股票" | 自然语言文本 |
| **suggestions.equity** | 建议把多少比例的钱买股票 | 直接告诉你：现在该拿多少钱买股票（剩下的是债券/黄金/现金） | 0-100 的整数 |
| **suggestions.bond** | 建议把多少比例的钱买债券 | 同上 | 0-100 的整数 |
| **suggestions.gold** | 建议把多少比例的钱买黄金 | 同上 | 0-100 的整数 |
| **suggestions.cash** | 建议留多少比例的现金 | 现金就是"子弹"，留着等机会再打出去 | 0-100 的整数 |
| **factor_details** | 每个因子的详细打分 | 就像成绩单上各科的成绩，能看明白总分是怎么来的 | 对象 |


### 二、各因子打分（factor_details）—— 总分的"明细账"

每个因子都包含以下字段：

| 子字段 | 含义 | 打个比方 |
|---|---|---|
| **signal** | 这个因子给宏观环境打的分 | **+2**=非常支持股市，**+1**=支持，**0**=无所谓，**-1**=拖累，**-2**=严重拖累 |
| **weight** | 这个因子在总分里占多大的比重 | 英语课占期末总成绩30%，这个数就是30% |
| **contribution** | 这个因子最终为总分贡献了多少分 | signal × weight = contribution，正分就是"加分项"，负分就是"减分项" |


### 三、各因子的具体含义

#### ① 中美利差信号（cn_us_spread）

| 概念 | 解释 |
|---|---|
| **它是什么？** | 中国10年期国债收益率 减去 美国10年期国债收益率。简单说就是"在中国买长期国债，比在美国买，每年多赚/少赚多少利息" |
| **为什么重要？** | 全球资金会流向利息更高的地方。如果中美利差**收窄**（倒挂减少），外资会更愿意来中国买债买股 |
| **正分代表什么？** | 利差在收窄 → 外资回流中国 → 利多A股 |
| **负分代表什么？** | 利差在扩大（倒挂加深）→ 资金外流 → 利空A股 |
| **小白一句话** | "中美利差就像两国存款利率的差价。差价在缩小，外国人的钱就更愿意来中国。" |


#### ② M1-M2剪刀差信号（m1_m2_spread）

| 概念 | 解释 |
|---|---|
| **它是什么？** | M1是企业手里的"活钱"（随时能花的钱），M2是"活钱+定期存款+储蓄"。**剪刀差 = M1增速 - M2增速** |
| **为什么重要？** | 如果剪刀差在**收窄**（负得少了），说明企业愿意把定期存款取出来花——这是经济活跃的信号 |
| **正分代表什么？** | 剪刀差收窄 → 企业敢花钱、敢投资 → 利多A股 |
| **负分代表什么？** | 剪刀差走阔 → 企业躺平、钱都存定期 → 利空A股 |
| **小白一句话** | "M1-M2剪刀差就像企业的'花钱意愿指数'。收窄说明企业舍得花钱了，经济在变好。" |


#### ③ VIX信号（vix）

| 概念 | 解释 |
|---|---|
| **它是什么？** | 美国股市的"恐慌指数"，数值越高表示股民越恐慌 |
| **为什么重要？** | 全球资金的风险偏好是联动的。美股恐慌，全球资金都会收缩，A股也会受影响 |
| **正分代表什么？** | VIX低（<18）→ 市场不恐慌 → 利多A股 |
| **负分代表什么？** | VIX高（>25）→ 市场恐慌 → 利空A股 |
| **小白一句话** | "VIX就是全球股民的心跳。心跳平稳说明大家都很淡定，心跳加速说明有人要逃命。" |


#### ④ 北向资金信号（north_money）

| 概念 | 解释 |
|---|---|
| **它是什么？** | 通过香港交易所买入A股的外资，每天公布净买入/净卖出的金额 |
| **为什么重要？** | 外资被称为"聪明钱"，他们的买入卖出往往领先于A股涨跌 |
| **正分代表什么？** | 连续净流入 → 外资看好A股 → 利多 |
| **负分代表什么？** | 连续净流出 → 外资撤退 → 利空 |
| **小白一句话** | "北向资金就是外资在A股买的'购物车'。购物车越装越满，说明外资觉得A股便宜；越倒越空，说明他们要跑。" |


#### ⑤ 信用利差信号（credit_spread）

| 概念 | 解释 |
|---|---|
| **它是什么？** | 高风险公司债券的利率 减去 国债利率。差值越大，说明企业借钱越难、越贵 |
| **为什么重要？** | 这是**系统性风险的预警器**。信用利差急扩，往往比股市暴跌来得更早 |
| **正分代表什么？** | 利差低且稳定 → 企业借钱容易 → 无系统性风险 |
| **负分代表什么？** | 利差急扩（>600bp）→ 信用冻结 → 可能是金融危机前兆 |
| **小白一句话** | "信用利差就像企业的'借钱困难指数'。利差突然暴涨，说明企业可能借不到钱了——这是危险信号。" |


### 四、中国内核模块（china_core）—— 中国经济的"体检报告"

| 字段 | 含义 | 小白解释 |
|---|---|---|
| **pmi** | 官方制造业采购经理指数 | **经济的"体温计"**。>50=经济在发热（扩张），<50=在发冷（收缩）。每月第一个公布，最领先 |
| **cpi_yy** | 居民消费价格同比涨幅 | **物价的"涨速表"**。<1%=东西太便宜了（通缩），>3%=东西太贵了（通胀）。1-2%最舒服 |
| **ppi_yy** | 工业品出厂价格同比涨幅 | **工厂产品"卖得好不好"**。负值=工厂打折也卖不动（利润承压），正值=工厂能涨价（利润改善） |
| **m1_m2_spread** | M1增速 - M2增速 | **企业的"花钱意愿"**。收窄=企业敢花钱，走阔=企业躺平（见上文详细解释） |
| **social_financing_yoy** | 社融存量同比增速 | **实体经济"拿到多少钱"**。增速回升=企业拿到钱了（信用扩张），回落=企业拿不到钱（信用收缩） |
| **lpr_1y** | 1年期贷款市场报价利率 | **企业贷款的"基准利率"**。下调=降息（利好），上调=加息（利空） |


### 五、全球宏观模块（global_macro）—— 外部世界的"天气预报"

| 字段 | 含义 | 小白解释 |
|---|---|---|
| **us_ism_pmi** | 美国制造业PMI | **全球经济的"风向标"**。>50=美国经济好（利好全球），<50=美国经济差（拖累全球） |
| **us_core_pce_yy** | 美国核心PCE同比 | **美联储的"降息开关"**。>3%=通胀太高，不能降息（利空全球）；<2.5%=可以降息了（利好全球） |
| **fed_balance_sheet** | 美联储资产负债表规模 | **全球资金的"水龙头"**。扩表=放水（利多），缩表=抽水（利空） |
| **on_rrp_balance** | 隔夜逆回购规模 | **市场里"闲钱"的多少**。钱多时这个数大，钱少时这个数小。接近0=流动性可能出问题 |


### 六、美元与流动性模块（fx_liquidity）—— 汇率的"水位计"

| 字段 | 含义 | 小白解释 |
|---|---|---|
| **dxy** | 美元指数 | **美元是"强"还是"弱"**。>105=美元太强（压制A股和黄金），<100=美元弱（利好A股和黄金） |
| **cn_us_10y_spread** | 中美10年期国债利差 | 见上文"中美利差"详解 |
| **usd_cnh** | 离岸人民币汇率 | **1美元需要多少人民币**。>7.30=人民币在贬值（利空A股），<7.10=人民币在升值（利多A股） |
| **usd_jpy** | 美元兑日元汇率 | **全球套息交易的"温度计"**。日元急升=套息平仓（可能引发全球流动性危机） |


### 七、全球核心资产模块（assets）—— 各类资产的"价格表"

| 字段 | 含义 | 小白解释 |
|---|---|---|
| **spx** | 标普500指数 | **美股"温度计"**。美股涨=全球风险偏好高，美股跌=全球在避险 |
| **hs300** | 沪深300指数 | **A股大盘"成绩单"**。中国最大的300家公司，代表A股整体表现 |
| **hsi** | 恒生指数 | **港股"晴雨表"**。对全球流动性特别敏感，美元强时港股通常承压 |
| **us_10y_yield** | 10年期美债收益率 | **全球利率的"天花板"**。这个利率高=全球借钱都贵（利空成长股），低=全球借钱便宜（利多成长股） |
| **gold_spot** | 黄金现货价格 | **避险的"硬通货"**。地缘风险+实际利率下降时会上涨 |
| **wti_oil** | WTI原油价格 | **全球通胀的"油门"**。油价>90美元=通胀压力大（利空），<70美元=通胀压力小（利好） |


### 八、风控模块（risk）—— 市场的"警报器"

| 字段 | 含义 | 小白解释 |
|---|---|---|
| **hy_spread** | 高收益债信用利差 | **企业"借钱难易度"**。>600bp=危险（可能爆发危机），<450bp=安全（一切正常）。这是比VIX更早的预警信号 |
| **move_idx** | MOVE债券波动率 | **债券市场的"恐慌指数"**。债券市场如果剧烈波动，往往预示着更大的风暴 |
| **cn_vix** | A股隐含波动率 | **A股的"恐慌指数"**。>35=A股在恐慌性抛售（可能见底），<15=过于平静（警惕变盘） |


### 九、事件流模块（events）—— 接下来可能"引爆"市场的事情

| 字段 | 含义 | 小白解释 |
|---|---|---|
| **title** | 事件名称 | 比如"美国CPI数据发布"、"美联储利率决议" |
| **risk_level** | 风险等级 | 🔴高风险=可能引发市场大幅波动，🟡中风险=有影响但不致命，🟢低风险=看看就好 |
| **time** | 发布时间 | 北京时间，到时可以提前蹲守 |
| **impact_assets** | 影响哪些资产 | 比如"黄金"、"A股"、"人民币"，告诉你要盯住什么 |

## 📌 十、快速理解表：5个信号分别对应什么？

如果你只想记住最重要的事情，记住这**5个"信号灯"**就够了：

| 信号灯 | 代表什么 | 绿灯（好） | 红灯（危险） |
|---|---|---|---|
| 中美利差 | 外资流不流入 | 收窄中 | 持续走阔 |
| M1-M2剪刀差 | 企业花不花钱 | 收窄中 | 持续走阔 |
| VIX | 全球恐不恐慌 | < 18 | > 28 |
| 北向资金 | 外资买不买 | 持续流入 | 持续流出 |
| 信用利差 | 系统安不安全 | < 450bp | > 600bp |
