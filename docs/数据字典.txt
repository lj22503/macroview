### ⚙️ Agent自动化部署极简版（给开发者的备注）

基于以上字典，Agent只需要对接**3个核心数据管道**即可拉取95%以上的字段：

1. **FRED API**（美联储经济数据）：
   - 覆盖字段：`VIXCLS`, `MOVE`, `DGS10`, `DGS2`, `PCEPILFE`, `CPIAUCSL`, `WALCL`, `RRPONTSYD`, `FEDFUNDS`, `SP500`, `BAMLH0A0HYM2`, `TEDRATE`, `CBBTCUSD`, `DTWEXBGS`
   - 接入成本：免费（需在FRED官网注册API Key，一分钟搞定）

2. **AKShare**（国产开源金融数据库）：
   - 覆盖字段：所有中国宏观数据（PMI、CPI、社融、M2、LPR）、北向/南向资金、沪深300、恒生指数、中债收益率、USD/CNH
   - 接入成本：`pip install akshare`，零配置

3. **yfinance**（雅虎财经）：
   - 覆盖字段：全球股指实时价格、黄金(`GC=F`)、原油(`CL=F`)、外汇对(`USDCNH=X`)
   - 接入成本：`pip install yfinance`，零配置

**数据流时序设计建议**：
- **每日 08:30**：Agent 唤醒，拉取 FRED 日频数据 + 刷新 AKShare 国内宏观
- **盘中 (每 5 分钟)**：仅轮询 yfinance 实时价格（股指、汇率、商品）
- **每月 20 日/月末**：Agent 重点校验 LPR 和 PMI 的发布窗口

这份字典可以直接作为你数据库建表（MySQL/MongoDB）的 Schema，或者作为 Agent Function Call 的入参定义。需要我把这份字典导出成 **JSON Schema** 或 **SQL DDL（建表语句）** 吗？方便你直接开始写代码。
